# JobHub Pro Backend

Backend service for JobHub Pro, a production-style job portal application that connects job seekers and recruiters.

## Tech Stack

- Java 21
- Spring Boot
- Spring Security
- JWT Authentication
- Hibernate / JPA
- MySQL
- Maven

## Features

- User Authentication
- Role-Based Authorization
- Company Management
- Job Management
- Application Tracking
- Interview Scheduling
- Notifications
